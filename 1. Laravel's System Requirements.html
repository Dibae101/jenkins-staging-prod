For you to develop Laravel applications on your computer, you need to have the following programs installed:

  • PHP: The programming language that Laravel is built on,

  • Any relational database of your choice but MySQL is used in this course,

  • Composer which is a dependencies manager for PHP applications.

Laravel Homestead is good for this as it gives you everything you need to develop any PHP application out of the box. But it is a bit tricky to get started with if you are new to developing PHP/Laravel applications.

If you want a quick start, you may consider XAMPP which is the program used in this course. There are other options but I think any of these is good enough to get started with.

If you go with XAMPP, you will need to install Composer separately on your machine. Click "Getting Started" for the installation process.
